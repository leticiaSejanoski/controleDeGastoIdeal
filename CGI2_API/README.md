# 💰 Controle De Gasto Ideal
Sistema de controle de gastos desenvolvido em equipe para a aula de **Tópicos Especiais de Sistemas** em **C#**, utilizando conceitos básicos de **POO** e **WebApi**.

Este projeto tem como objetivo ajudar usuários a **registrar, organizar e analisar seus gastos pessoais**, permitindo um melhor controle financeiro.

## 🚀Funcionalidades:
- 📁 Cadastrar categoria;
- 💸 Cadastrar gasto;
- 📋 Listar categorias;
- 📃 Listar gastos;
- 📊 Consultar total de gastos por categoria;
- 📈 Consultar total de gastos;
- ❌ Remover categoria;
- 🗑️ Remover gasto;
- 🔄 Atualizar gastos;
- ✏️ Atualizar categorias.

 ---

## 📏Regras de Negócio
- 📝 Impedir nome de categoria vazio;
- 🔁 Impedir categorias com nomes duplicados;
- 📃 Impedir descrição de gasto vazia;
- 💲 Validar valor do gasto (deve ser maior que zero);
- 🔗 Todo gasto deve estar vinculado a uma categoria;
- 🔍 Verificar se a categoria informada existe;
- 🚫 Impedir cadastro de gastos quando não houver categorias cadastradas.
