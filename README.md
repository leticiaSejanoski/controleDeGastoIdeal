# 💰 Controle De Gasto Ideal
Sistema de controle de gastos desenvolvido em equipe para a aula de **Tópicos Especiais de Sistemas** em **C#**, utilizando conceitos básicos de **POO**.

Este projeto tem como objetivo ajudar usuários a **registrar, organizar e analisar seus gastos pessoais**, permitindo um melhor controle financeiro.

## 🚀Funcionalidades:
- 📁Cadastrar categoria;
- 💸Cadastrar gasto;
- 📋Listar categorias;
- 📃Listas gastos;
- 📊Total por categoria;
- 📈Total de gastos;
- ❌Remover categoria;
- 🗑Remover gasto;
- 🚪Sair.

 ---

## 📏Regras de negócios:
- ❗Impedir lista vazia;
- 📝Impedir nome vazio;
- 🔁Impedir duplicação de categoria;
- 💲Validar valor(não negativo);
- 📃Validar descrição;
- 🔢Evitar a inserção de dados não numéricos em campos numéricos.
